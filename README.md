## ব্যবহার

### স্থানীয়ভাবে চালানো:
1. `index.html` ফাইলটি ব্রাউজারে খুলুন
2. নেভিগেশন মেনু থেকে আপনার প্রয়োজনীয় পেজে যান
3. ধাপে ধাপে নির্দেশনা অনুসরণ করুন

### GitHub Pages এ ডিপ্লয় করা:

#### প্রথমবারের জন্য:
1. GitHub এ একটি নতুন repository তৈরি করুন
2. Repository name: `makkahmadinah` (বা আপনার পছন্দমতো)
3. Repository visibility: Public (GitHub Pages এর জন্য প্রয়োজন)
4. নিচের কমান্ডগুলো রান করুন:

```bash
# Repository initialize করুন (যদি এখনও না করা হয়ে থাকে)
git init

# সব ফাইল add করুন
git add .

# প্রথম commit করুন
git commit -m "Initial commit: Umrah & Hajj Guide Website"

# GitHub repository যোগ করুন (YOUR_USERNAME পরিবর্তন করুন)
git remote add origin https://github.com/YOUR_USERNAME/makkahmadinah.git

# Main branch এ push করুন
git branch -M main
git push -u origin main
```

#### GitHub Pages সক্রিয় করা:
1. GitHub repository তে যান
2. **Settings** ট্যাবে ক্লিক করুন
3. বাম সাইডবারে **Pages** সেকশনে যান
4. **Source** এর নিচে **Deploy from a branch** নির্বাচন করুন
5. Branch: **main** (বা **master**) এবং folder: **/ (root)** নির্বাচন করুন
6. **Save** ক্লিক করুন
7. কয়েক মিনিট অপেক্ষা করুন, তারপর আপনার website এখানে available হবে:
   `https://YOUR_USERNAME.github.io/makkahmadinah/`

#### পরবর্তী আপডেটের জন্য:
```bash
git add .
git commit -m "Update: Description of changes"
git push origin main
```

GitHub Actions automatically আপনার website deploy করবে!

## পেজসমূহ

### হোমপেজ (index.html)
- ওয়েবসাইটের মূল পেজ
- বিভিন্ন সেকশনের সংক্ষিপ্ত বিবরণ
- দ্রুত নেভিগেশন

### উমরাহ (umrah.html)
- ইহরাম গ্রহণ থেকে শুরু করে সম্পূর্ণ উমরাহ প্রক্রিয়া
- প্রতিটি ধাপের বিস্তারিত নির্দেশনা
- প্রয়োজনীয় দোয়া ও সূরা

### হজ্জ (hajj.html)
- ৮ই জিলহজ্জ থেকে শুরু করে সম্পূর্ণ হজ্জ প্রক্রিয়া
- দিনে দিনে বিস্তারিত নির্দেশনা
- আরাফাত, মুজদালিফা, মিনা সম্পর্কে তথ্য
- রমি (পাথর মারার) প্রক্রিয়া

### দোয়া ও সূরা (duas.html)
- উমরাহ ও হজ্জের জন্য সমস্ত প্রয়োজনীয় দোয়া
- আরবি, রোমানাইজেশন এবং বাংলা অনুবাদ
- বিভিন্ন সূরা

### গাইড (guide.html)
- প্রস্তুতি সম্পর্কে তথ্য
- ইহরাম সম্পর্কে বিস্তারিত
- মিকাত সম্পর্কে তথ্য
- গুরুত্বপূর্ণ পরামর্শ
- বাংলাদেশী হাজিদের জন্য বিশেষ পরামর্শ
